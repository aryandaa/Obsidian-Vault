#programming
Untuk melakukan berbagai perhitungan pada python
```Python
a = 15
b = 3

#tambah
tambah = a + b
print(tambah)

#kurang
kurang = a - b
print(kurang)

#pembagian
bagi = a / b
print(bagi)

#perkalian
kali = a * b
print(kali)

#pangkat (eksponen)
pangkat = a ** b
print(pangkat)

#hasil bagi (modulus)
modulus = a % b
print(modulus)

#floor division
floor_division = a // b
print(floor_division)
```
