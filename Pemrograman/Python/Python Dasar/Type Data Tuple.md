#programming 
type data tuple hampir mirip dengan list, bedanya adalah tuple tidak dapat di ubah atau immutable, dan juga tuple menggunakan tanda kurung (). 
karena tuple tidak bisa di ubah maupun di hapus datanya, kelebihan dari list adalah proses tuple lebih cepat dibandingkan list.

Cara menggunakannya cukup simple:
```python
warna = ("biru", "hijau", "merah")
print (warna)
#biru, hijau, merah
```
ketika kita mencoba mengubah value nya, maka code akan error.
