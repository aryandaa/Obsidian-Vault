#programming 
count() method dalam python digunakan untuk menghitung jumlah element dalam object/variable, metode ini juga bisa digunakan untuk mencari element tertentu di dalam object.
contohnya code dibawah ini:

list:
```python
list1 = [11,11,11,22,33,44] 
print(list1.count(11)) # mencari total angka 11 dalam list

#output
3
```

tuple:
```python
tup1 = (11,11,11,22,33,44) 
print(tup1.count(11)) # mencari total angka 11 dalam tuple

#output
3
```

Strings
```python
str1 = 'hello world'
print(str1.count('l')) # mencari total huruf l dalam string

#output
3
```
