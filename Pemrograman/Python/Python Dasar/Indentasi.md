#programming 
di dalam python, indentasi sangatlah penting untuk menyatakan kepemilikan dari sebuah aksi, sebagai contoh:
```python
if a > b:
    print("c")
```
dari code diatas kita tau kalo aksi print("c") itu milik function if. 

jika kita tidak menggunakan indentasi. maka print("c") dianggap tidak memiliki function dan akan menjadi function pemanggil string "c" biasa oleh sistem
