#programming 
For:
```js
for(let i=0; i<5; i++){
    console.log("Hello World");
}
 
console.log("Sudah di luar kondisi for");
```
disini code akan mencetak 5 kali "Hello world"

While:
```js
let counter = 0;
 
while(counter < 5){
    console.log("Hello World");
    counter++;
}
 
console.log("Sudah di luar kondisi while");
```
