#programming 
Pada modul sebelumnya, kita telah mempelajari bagaimana BOM dan DOM berperan terhadap halaman web. Kita dapat memanipulasi konten halaman web menggunakan JavaScript. Bahkan, kita juga dapat memanipulasi elemen HTML, seperti menghapus, menambahkan, hingga memanipulasi atributnya.

Namun, pernahkah Anda terpikirkan cara agar suatu elemen dapat interaktif ketika mendapatkan suatu feedback (event)? Sebagai contoh, melakukan suatu hal ketika elemen tombol ditekan. Nah, dalam modul ini, kita akan mempelajari penerapan Event pada JavaScript.

Pada akhir modul ini, Anda diharapkan dapat:

- Mengetahui Event beserta macam-macamnya.
- Mampu menambahkan Event Handler pada HTML Element.
- Mengetahui Custom Event serta mampu membuat dan menggunakannya.
- Mengetahui konsep dari Event Bubbling dan Event Capturing.
- Mengetahui Event yang berada pada elemen `<form>`.
- Mengetahui Event yang berada pada elemen`<input>`.
