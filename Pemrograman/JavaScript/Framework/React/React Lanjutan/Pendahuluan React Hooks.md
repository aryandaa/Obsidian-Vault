#programming 
Saat ini, React berada di fase baru. Seiring penggunaan yang melejit, tim React ingin menciptakan _developer experience_ yang sama baiknya dengan popularitasnya. Dengan menciptakan developer experience yang baik, developer diharapkan dapat lebih produktif untuk menciptakan inovasi baru, baik di lingkup ekosistem React maupun produk digital yang dikembangkan. 

Untuk mencapai tujuan tersebut, React menghadirkan fitur baru bernama hooks. Hooks membawa banyak perbaikan untuk menyelesaikan masalah atau titik lemah pada React yang ada saat ini. Dengan hooks, kita bisa mendapatkan banyak benefit mulai dari menyederhanakan kode, menciptakan kode yang reusable, hingga membuat developer dan klien bahagia.

Modul ini akan membahas banyak hal terkait React Hooks. Berikut beberapa poin pembahasannya.

- Mengetahui alasan hadirnya React Hooks.
- Mengenal masalah dan titik lemah React yang akan diselesaikan oleh React Hooks.
- Mengelola state dengan fungsi `useState()`.
- Membuat efek dengan fungsi `useEffect()`.
- Mendapatkan nilai React Context dengan fungsi `useContext()`.
- Mengekstraksi dan membagikan logika nonvisual dengan custom hooks.
- Mengimplementasikan React Hooks pada proyek studi kasus aplikasi kontak.
