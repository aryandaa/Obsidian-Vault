#programming 
Ketika menggunakan aplikasi web atau sekadar berselancar di internet, Anda pasti sadar bahwa mayoritas website dapat menampilkan lebih dari satu halaman. Secara alami, website mengandalkan URL untuk menampilkan halaman yang berbeda.

Namun, ada tantangan tersendiri bila kita menggunakan React dalam membuat aplikasi web. Karena seluruh antarmuka ditampilkan menggunakan JavaScript, bagaimanakah cara yang efektif untuk menampilkan halaman lebih dari satu? Apakah memanfaatkan state dan teknik conditional rendering saja cukup? Atau, ada hal lain yang perlu dipertimbangkan? Tenang, modul ini akan mengupas tuntas tentang penerapan routing di React yang efektif.

Berikut beberapa poin yang akan dibahas di modul ini.

- Memahami bagaimana routing bekerja pada aplikasi web.
- Latihan membuat routing sederhana pada aplikasi React dengan memanfaatkan _state_.
- Mengenal React Router sebagai salah satu ekosistem React untuk mengimplementasikan _routing_.
- Latihan menggunakan React Router untuk membuat routing sederhana pada aplikasi React.
- Mengenal teknik Path dan Query parameter untuk mengirimkan data melalui URL.
- Latihan menerapkan Path dan Query parameter menggunakan React Router.
- Latihan memisahkan halaman fitur tambah kontak dan membuat fitur pencarian pada proyek studi kasus.
- Mengetahui cara menggunakan React Icons pada proyek React.
