#programming 
Di React, komponen merupakan fondasi dalam membangun aplikasi seutuhnya. Dengan komponen, kita dapat mengelola state, mengirim dan menerima data melalui props, hingga melukiskan antarmuka kepada pengguna. Komponen di React memang luar biasa manfaatnya, tetapi semakin kompleks aplikasi yang dibangun, semakin kompleks pula kode yang dituliskan di dalam komponen.

Tugas berat seperti AJAX request atau proses _asynchronous_ lain sulit untuk dihindari dalam pembuatan aplikasi. Tugas tersebut secara tidak langsung akan menambah “tanggung jawab” komponen juga. Agar kode di dalam komponen tetap terkelola dengan baik, Anda harus tahu praktik yang tepat dalam menangani proses tersebut di komponen React.

Modul ini akan mengajarkan tentang Component Lifecycle sehingga Anda akan

- mengetahui cara aplikasi front-end menampilkan data dinamis;
- mengetahui RESTful API, fetch, dan same-origin policy;
- mengetahui pengertian component lifecycle;
- mengetahui fase mounting, updating, dan unmounting pada komponen;
- latihan menggunakan lifecycle komponen; dan
- mengimplementasikan lifecycle komponen pada proyek studi kasus.
