#programming 
logika AND adalah logika dimana output akan benilai 1/true jika kedua nilai input 1/true,
jika salah satunya bernilai 0/false, maka outputnya akan false.

jika digambarkan dengan table kebenaran:
1 & 1 = 1
0 & 1 = 0
1 & 0 = 0
0 & 0 = 0

itu penjelasan jika memiliki 2 inputan, bagaimana jika inputannya adalah 3 atau lebih? bisa, tetapi persyaratannya untuk mendapatkan true tetap belaku juga, yang artinya nilai harus true semua, jika ada satu saja yang false, maka akan membatalkan true.

logika AND jika di pakai dalam kehidupan sehari hari akan menjadi begini:
Anda akan mendapat diskon sebesar 20% jika memiliki kartu pengguna (member card) dan minimal belanja sebesar 100 ribu. Sehingga Anda harus memenuhi dua persyaratan tersebut untuk mendapat diskon. Jika Anda hanya memiliki member card tetapi total belanja masih di bawah 100 ribu, maka Anda tidak akan mendapat diskon. Begitu pun sebaliknya.
