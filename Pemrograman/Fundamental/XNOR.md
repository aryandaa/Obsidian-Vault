#programming 
gerbang logika terakhir yaitu XNOR yaitu singkatan dari Exclusive NOR, 
 Jika sebelumnya dalam logika NOR kita mengetahui jika salah satu input ada yang bernilai 1 (true), maka outputnya pasti bernilai 0 (false). 
 Lain halnya dengan XNOR yang hanya menghasilkan keluaran 1 (true) ketika kedua nilai inputnya sama, dan menghasilkan 0 (false), jika inputan berbeda.

contoh tabel kebenarannya:
1 xnor 1 = 1
0 xnor 1 = 0
1 xnor 0 = 0
0 xnor 0 = 1
