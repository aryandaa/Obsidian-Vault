## Apa itu laravel dan mengapa memilih laravel?
Laravel adalah framework PHP yang dirancang untuk khusus web, bukan untuk dekstop maupun mobile. 

Model View Controller (MVC)

### Instalasi
Persyaratan Sistem:
- XAMPP / Laragon
- Composer
- Visual Studio Code

Setelah berhasil meninstall 3 tools diatas, masuk ke folder htdocs (Xampp), atau www (laragon).
terus ketik di cmd `composer create-project laravel/laravel my-project-name`
